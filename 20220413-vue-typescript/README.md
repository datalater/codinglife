# Vue typescript 프로젝트 세팅

## Installation

```bash
yarn add vue@2.6.12 vue-router@3.2 vuex@3
yarn add vue-template-compiler@2.6.12
yarn add @vue/{composition-api,compiler-sfc}

yarn add typescript

yarn add -D webpack webpack-cli webpack-dev-server

yarn add -D babel-loader vue-loader@15 vue-style-loader css-loader copy-webpack-plugin html-webpack-plugin postcss postcss-loader autoprefixer sass sass-loader

yarn add -D @babel/{core,plugin-transform-runtime,preset-env,runtime-corejs3} babel-preset-typescript-vue

yarn add -D eslint eslint-plugin-vue
```
